# Task 002: 음성 샘플 녹음 가이드 작성

**상태:** 완료  
**담당:** Codex  
**의존성:** task-001 완료 권장  
**완료일:** 2026-03-18

## 목표

강의/실습 수강생이 XTTS v2 클로닝용 음성 샘플을 스스로 준비할 수 있도록 `docs/02-recording.md`를 작성한다.

## 입력 (Input)

- `docs/01-setup.md` (1단계 문서)
- 프로젝트 컨텍스트 (macOS + Apple Silicon)
- 교안 품질 기준(학습 목표, 단계별 구조, 실행 가능성, 트러블슈팅)

## 출력 (Output)

- `docs/02-recording.md` 신규 작성
- 녹음 권장 스펙, 단계별 절차, 품질 체크리스트 포함
- 다음 단계(`03-cloning.md`)로 연결 링크 포함

## 작업 단계

1. 문서 도입부에 학습 목표/난이도/예상 시간 명시
2. 녹음 준비물과 권장 스펙 표 작성
3. 단계별 녹음 절차(녹음→저장→포맷 정리→검증) 작성
4. 트러블슈팅 및 품질 체크리스트 추가
5. 이전/다음 단계 링크 추가

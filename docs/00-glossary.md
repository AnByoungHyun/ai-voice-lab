# 용어집 (Glossary)

이 프로젝트에서 자주 등장하는 기술 용어를 정리합니다. 처음 보는 용어가 나오면 이 문서를 참고하세요.

## 음성 / TTS 관련


| 용어                         | 설명                                             |
| -------------------------- | ---------------------------------------------- |
| **TTS**                    | Text-to-Speech. 텍스트를 음성으로 변환하는 기술              |
| **음성 클로닝 (Voice Cloning)** | 특정 화자의 음성 특징을 학습하여 해당 목소리로 새로운 텍스트를 읽어주는 기술    |
| **화자 (Speaker)**           | 음성의 주인. 클로닝 대상이 되는 사람                          |
| **레퍼런스 음성**                | 클로닝 시 참조하는 원본 음성 파일. XTTS v2는 최소 6초 분량이 필요하다   |
| **파인튜닝 (Fine-tuning)**     | 사전 학습된 모델을 특정 데이터(예: 내 음성)로 추가 학습시켜 품질을 높이는 과정 |


## 모델 / 프레임워크


| 용어               | 설명                                                                     |
| ---------------- | ---------------------------------------------------------------------- |
| **XTTS v2**      | Coqui AI가 개발한 다국어 음성 클로닝 모델. 6초 샘플로 화자 복제가 가능하며, 한국어를 포함한 17개 언어를 지원한다 |
| **Coqui TTS**    | 오픈소스 TTS 엔진. XTTS v2를 포함한 여러 TTS 모델을 통합 제공하는 Python 라이브러리              |
| **PyTorch**      | Meta(Facebook)가 개발한 딥러닝 프레임워크. 이 프로젝트의 모든 모델이 PyTorch 기반으로 동작한다        |
| **transformers** | Hugging Face의 모델 라이브러리. XTTS v2 내부에서 언어 모델 컴포넌트를 로드할 때 사용한다            |
| **torchcodec**   | PyTorch 2.9+에서 오디오/비디오 인코딩·디코딩을 처리하는 라이브러리                             |
| **librosa**      | Python 오디오 분석 라이브러리. 음성 파일의 샘플레이트, 길이, 스펙트로그램 등을 다룬다                   |


## 하드웨어 / 환경


| 용어                | 설명                                                                                         |
| ----------------- | ------------------------------------------------------------------------------------------ |
| **Apple Silicon** | Apple이 자체 설계한 ARM 기반 칩 (M1, M2, M3, M4, M5 시리즈). GPU 가속을 내장하고 있다                           |
| **MPS**           | Metal Performance Shaders. Apple Silicon GPU를 PyTorch에서 사용하기 위한 백엔드. NVIDIA의 CUDA에 해당하는 역할 |
| **CUDA**          | NVIDIA GPU에서 병렬 연산을 수행하기 위한 플랫폼. Linux 또는 Windows 환경에서 사용                                  |
| **venv**          | Python 내장 가상환경 도구. 프로젝트별로 독립된 패키지 환경을 만들어 의존성 충돌을 방지한다                                     |


## 서비스 / 플랫폼


| 용어               | 설명                                                  |
| ---------------- | --------------------------------------------------- |
| **OpenClaw**     | AI 어시스턴트 플랫폼. 커스텀 TTS를 연동하여 나만의 음성 AI를 만들 수 있다      |
| **Hugging Face** | ML 모델과 데이터셋을 공유하는 플랫폼. XTTS v2 모델 가중치가 여기에 호스팅되어 있다 |
